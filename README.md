# NPC Assistant Preview

This project was automatically set up to preview the `NPC_Generator_Chatbot.jsx` component.

## Running the Project

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Open the browser**:
    Navigate to [http://localhost:5173/](http://localhost:5173/).

## Configuration

To make the app fully functional, you need to provide:

1.  **Firebase Configuration**:
    Open `index.html` and update the `window.__firebase_config` variable with your Firebase project configuration.

2.  **Gemini API Key**:
    Open `src/NPC_Generator_Chatbot.jsx` and set the `apiKey` variable at the top of the file.

## Project Structure

- `src/NPC_Generator_Chatbot.jsx`: The main application component.
- `src/main.jsx`: Entry point.
- `index.html`: HTML template with global config placeholders.
